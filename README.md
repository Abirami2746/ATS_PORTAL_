This project is a **Mini job portal** built using **HTML, CSS, JavaScript**, and a **Java backend (Servlets + DAO)**.  
It simulates how real job portals work by allowing job applicants to upload resumes, apply for jobs, calculate ATS scores, and track applications.
### Job applicant Module
- User Registration & Login
- Upload Resume
- View Available Jobs
- Apply for Jobs (with resume)
- Save / Bookmark Jobs
- View Applied Jobs
- ATS Score Calculation (Resume vs Job Description)
- Missing Skills Identification
- Suggested Courses based on ATS score
- Profile Update with Photo
- Application Status Tracking
